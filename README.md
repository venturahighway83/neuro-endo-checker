Neuro-Endo Checker（脳血管内治療デバイス適合チェッカー）

脳血管内治療に用いるカテーテル／デバイスの“適合性”を可視化・比較するWebアプリです。
径・長さ・組合せの整合性を直感的に確認でき、TRA/DRAアプローチやFD導入ルート検討を補助します。

Live: https://neuro-endo-checker-jl5j-five.vercel.app
